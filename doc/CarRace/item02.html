ダウンロード版パッケージ

(2011,11/02)遅くなってしまいましたが，先日質問に来てくれた人のPCでJava Web Start版のプログラムが起動しない問題が発生したので，ダウンロード版を急遽公開します．Githubのサーバの容量がこころもとないため，学内のサーバに置いてあります．学外からアクセスするためにはVPN接続を利用して下さい．

  • Windows 32ビット版: CarRace32-20111102.zip
  • Windows 64ビット版: CarRace64-20111102.zip

Windowsが32ビットか64ビットかによって異なるので，自分の使用しているPCを調べて適切な方をダウンロードして下さい．Java6またはJava7のSDKをインストールしPATH環境変数を正しく設定してから，解凍すると出てくるCarRace.batをダブルックリックして起動してみて下さい．

MacOSX版は用意しませんでしたが，必要だという人は連絡下さい．

不明な点などあればプログラミングコンテスト実行委員procon@do-johodai.ac.jp または，メーリングリストprocon2011@ks1.orgにて連絡下さい．
